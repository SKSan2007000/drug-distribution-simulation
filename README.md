# Drug Distribution Simulation Capstone Project

## Title
Simulation and Analysis of Drug Distribution in the Human Body using Transform-Based Mathematical Modeling

## How to Run
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
Open: http://127.0.0.1:8000/
